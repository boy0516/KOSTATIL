```
Map<String,String> map = new HashMap<String,String>();
```



- 자료삽입

```
map.put("1","KT");
map.put("2","LG");
map.put("3","sam");
```



- 해시map안의 자료 여부

```
if(map.containsValue("LG")) {
	map.remove("2");
}
```
